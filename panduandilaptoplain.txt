  LANGKAH 3 â€” KONFIGURASI CONNECTION STRING (âš  CRITICAL)
================================================================================

  File yang harus diedit:
  â†’ backend/appsettings.json

  3.1  Buka file tersebut di text editor / VS Code.

  3.2  Cari bagian "ConnectionStrings". Isinya akan seperti ini:

         "MasterConnection":   "Server=ADAM123\\SQLEXPRESS;Database=db_master_data;...",
